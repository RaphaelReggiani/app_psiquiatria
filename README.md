# GMP

Aplicação para um site de gerenciamento de médicos psiquiatras e pacientes.

Desenvolvida com **Python**, **Django**, **Django REST Framework** e **MySQL** (Inclui API com autenticação via token para acesso seguro aos dados), utilizando **HTML** e **CSS**, para **UX**. 

A aplicação configura-se como um site voltado para uma empresa de fornecimento de serviços de psiquiatria, com cadastramento de usuário e membros da equipe (médicos), além do controle da STAFF (SuperADM's) e gerenciamento / controle de consultas entre psiquiatras e pacientes, assim como os registros / LOGs de todas as ações realizadas na aplicação.
